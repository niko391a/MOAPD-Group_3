## 📱 About the Project

**X9** is an Android application developed for the *Mobile App Development* course. 


### ✨ Current Key Features
*   **Incident Logging:** Report accidents, congestion, or road hazards instantly.


## 🛠️ Current Tech Stack

*   **Language:** Kotlin
*   **Build System:** [Gradle (Kotlin DSL)](build.gradle.kts)
*   **UI:** Jetpack Compose / XML Layouts

## 🚀 Getting Started

### Prerequisites

*   Android Studio
*   JDK 17 or newer

### Installation

1.  Clone the repo:
2.  Open the project in Android Studio.
3.  Sync Gradle files to download dependencies.
4.  Run the app on an emulator or physical device via the [`app`](app) configuration.
